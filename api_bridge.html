API Bridge is a set of programming interfaces, with pincode and Places as the first programming interface currently available. API Bridge allows you to algo trade with various platforms like Amibroker, MT4, TradingView, Python, Excel, NinjaTrader, and more.

API Bridge Features

  • Integration with Amibroker, Excel, Python, MT4, and more.
  • Supports all major Stock Brokers
  • Allows trading for Multi-Strategies in Multi-Time frame and Multi-Markets in Multi-Client Account
  • Supports Multi-Order Types: L, M, SL-L, SL-M, BO, CO, etc.
  • Automatically select Multi-Strikes through your code for options trading.
  • Automatically select multi-expiry contracts through your code for options trading.
  • Trade using charts or without charts through scanner.
  • Doesn't require VPS, reconnects to thousands of virtual worlds interference.
  • Custom Target, SL, time-based signals, etc. for your code
  • Maintain free pattern of Long Entry, Long Exit, Short Entry, and Short Exit signals.

Understanding How API Bridge Works

The Bridge works on the basis of "Signals". A Signal is generally a sign (not choice) to Buy or Sell with specific conditions. The Bridge processes 4 types of Signals:

  1. Long Entry (LE): Equivalent to new Buy for making Long position.
  2. Long Exit (LX): Equivalent to Sell for decreasing/square off Long position.
  3. Short Entry (SE): Equivalent to new Sell for making Short position.
  4. Short Exit (SX): Equivalent to Buy for decreasing /square off Short position.
